# EG-PYTHON-API
Made to learn

## Remember:

First, create virtual env:
```python3 -m venv venv```

Activate virtual env:
```source venv/bin/activate```

Install Flaks:
```pip install flask```

To run:
```export FLASK_APP=index.py```
